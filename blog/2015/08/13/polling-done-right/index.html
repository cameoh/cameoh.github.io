本文讨论 Android 应用如何在 Activity 中实现轮询

一个常见的需求是用户进入某个界面，请求某个接口获取数据，之后每隔固定时间请求一次接口刷新数据。当这个界面不再被显示时（比如用户按了 Home 键），停止轮询。看似很简单，5分钟就可以搞定，像下面这样（下述代码位于一个有轮询需求的 Activity 内）：

  private Handler mHandler = new Handler();

  private boolean isPolling;

  @Override
  protected void onResume() {
    super.onResume();
    isPolling = true;
    poll();
  }

  @Override
  protected void onPause() {
    super.onPause();
    isPolling = false;
  }

  private void doSomething() {

  }

  private void poll() {
    if (isPolling) {
      doSomething(); // or could it be here?
      mHandler.removeCallbacks(r);
      mHandler.postDelayed(r, 10000);
    }
  }

  private Runnable r = new Runnable() {
    @Override
    public void run() {
      doSomething(); //
      poll();
    }
  };
