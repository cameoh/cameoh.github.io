一些产品的想法

移动端（非游戏）

  • 提醒下车
  • 检测隐私泄露
  • 证明你付出了多少无谓时间，用于设立门槛（作用类似于验证码）
  • 证明在某时某地做了某事
  • 和好
  • 双胞胎社区
  • 对以前的自己说
  • 测试三观
  • 零食review
  • 定向搜索，精确限定
  • 社会化的读书笔记
  • 识别左右手的 SDK，重力加手势
  • 邀请码社交

游戏

  • 玩火
  • 扫地
  • 堵车

软件工程

  • 查看每一行代码的生命周期，统计平均值
  • 大数据语义分析，发十大贴
  • 自动生成 license 说明
  • 一键修改 github 错别字（chrome 插件）

其他

  • 知其所以然的社区，比如为什么 GUI 库要设计成单线程；写代码为什么要限制行宽；为什么要规定修饰词的顺序。
